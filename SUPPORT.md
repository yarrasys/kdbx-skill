# Support

Need help with a skill in this repo?

- 📖 Start with the [README](README.md) and the skill's own docs under [`skills/<name>/`](skills)
  (its `SKILL.md`, `README.md`, and `references/`).
- 🐛 **Bugs / feature requests:** open a [GitHub Issue](https://github.com/yarrasys/skills/issues)
  (pick the skill in the issue form).
- 💬 **Questions / ideas:** use [GitHub Discussions](https://github.com/yarrasys/skills/discussions)
  (or an issue if Discussions isn't enabled).
- 🔒 **Security vulnerabilities:** do **not** open a public issue — see [SECURITY.md](SECURITY.md)
  (private reporting, or **hello@yarrasys.com**).
- ✉️ **Other:** hello@yarrasys.com

Open-source software provided as-is under the MIT License; support is best-effort.
